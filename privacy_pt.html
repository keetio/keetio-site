Política de Privacidade

Levamos a sua privacidade a sério. Esta política descreve como o Keetio lida com os dados dos usuários.

  • Coleta de Dados: Apenas coletamos os dados necessários para operar o aplicativo.
  • Uso de Dados: Os dados são usados exclusivamente para fornecer funcionalidades do aplicativo.
  • Compartilhamento: Não compartilhamos seus dados com terceiros, exceto quando exigido por lei.
  • Segurança: Implementamos medidas para proteger seus dados.

Ao utilizar o Keetio, você concorda com esta política.
